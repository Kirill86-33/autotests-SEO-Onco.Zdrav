\# SEO-автотесты для проверки метатегов (title, description, keywords)



Этот репозиторий содержит автоматические тесты для проверки SEO-метатегов на страницах веб-портала. Тесты анализируют заголовки (title), мета-описания (description) и мета-ключевые слова (keywords) на соответствие заданным требованиям.

\## Описание

Проект предназначен для автоматизированной проверки SEO-параметров страниц. Он помогает убедиться, что все важные страницы портала имеют корректные метатеги, что влияет на поисковую оптимизацию и ранжирование.



Основные проверки:

\*   Наличие метатега `title` и его длина.

\*   Наличие метатега `description` и его длина.

\*   Наличие метатега `keywords`.

\*   Соответствие значений ожидаемым (по данным из CSV-файла).



\## Структура проекта

\*   `seo\_meta\_tags\_check.csv` – файл с эталонными значениями метатегов для проверяемых страниц (обычно содержит URL, ожидаемый title, description, keywords).

\*   `parse\_seo.py` – скрипт для сбора актуальных метатегов со страниц (может использоваться для обновления эталонов или для сравнения).

\*   `universal\_download.py` – вспомогательный скрипт для загрузки страниц или ресурсов.

\*   `admin.json` – возможно, конфигурационный файл с данными для авторизации (если требуется доступ к закрытым разделам).

\*   `sitemap.xml` – карта сайта, часто используется для получения списка всех URL для проверки.

\*   `.gitignore` – список файлов и папок, исключённых из версионирования.

\*   `README.md` – этот файл с описанием проекта.
